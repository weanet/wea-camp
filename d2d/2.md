## ✏️ 课后作业

声明一个 `WUED` 团队的对象数组，使用变量，函数，循环等 js 语法将最终结果渲染到 `index.html` 页面上。（注：不允许使用 Jquery）

``` js
var WUXD = [
  {
    name: '照照',
    age: 18
  },
  {
    name: '平江',
    age: 18
  },
  {
    name: '杨柳',
    age: 18
  },
  {
    name: '双双',
    age: 18
  },
  {
    name: '红雨',
    age: 18
  }
];
```
